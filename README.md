# 한국 vs 인도, 차량 스펙으로 중고차 적정가 맞히기 — 전통 ML vs 딥러닝

> **AI+X: Deep Learning · 최종 그룹 프로젝트 (Option A)**
> 두 시장(한국·인도)의 차량 스펙으로 중고차 가격을 예측하고, 전통 ML과 딥러닝을 비교한다.

## 개요

차량 스펙(연식·주행거리·연료·배기량 등)만으로 중고차 적정가를 예측하는 문제를 다룹니다.
"정확도"가 아니라 **단계별 분석 절차**를 보여주는 것이 목표입니다(Option A). 두 개의 비교 축으로 구성됩니다.

- **모델 축** — 전통 ML(선형회귀·랜덤포레스트·그래디언트 부스팅) vs 딥러닝(MLP 신경망)
- **시장 축** — 한국(DACON) vs 인도(CarDekho), *같은 방법론을 두 시장에 적용*

> ⚠️ 두 시장은 통화·가격 체계가 달라 가격(또는 RMSE)을 **직접 맞비교하지 않습니다.**
> 통화에 무관한 **R²**로는 "모델 순위·중요 변수 패턴이 시장 간에 비슷한지"를 보고,
> RMSE·MAE는 각 시장 *내부*의 모델 비교에만 씁니다.

## 데이터

| 시장 | 출처 | 규모 | 가격 단위 |
|---|---|---|---|
| 한국 | DACON 자동차 가격 예측 해커톤 (236114) | train 약 57,920행 | 백만원 |
| 인도 | Kaggle CarDekho `Car details v3.csv` | 약 8,100행 | INR(루피) |

데이터는 용량 문제로 저장소에 포함하지 않습니다. 받는 법은 **[data/README.md](data/README.md)** 참고.
두 데이터를 각각 `data/korea_cars.csv`, `data/india_cars.csv`로 저장하세요.

## 방법론 요약

1. **전처리** — 결측치(수치형 median), 파생변수 `car_age`, 타겟 로그 변환(`log1p`),
   범주형 인코딩, 80:20 분할, 수치형 표준화(선형·신경망용). 시장별 전처리는 분리하고 핵심 절차는 공유.
2. **모델 4종** — 선형회귀 / 랜덤포레스트 / 그래디언트 부스팅 / MLP 신경망 (두 시장 공통 적용).
3. **평가** — R²(시장 간 패턴 비교) · RMSE·MAE(시장 내 모델 비교).

자세한 계획은 **[기획안.md](기획안.md)**, 블로그 본문 초안은 **[블로그_초안.md](블로그_초안.md)** 참고.

## 실행 방법

```bash
pip install -r requirements.txt
# data/README.md 를 보고 CSV 2개를 data/ 에 저장한 뒤 실행
python used_car_price_prediction.py
```

- 실행하면 시장별 성능표(R²/RMSE/MAE)가 출력되고, 그래프가 `figures/`에 저장됩니다.
- 두 CSV 중 하나만 있어도 해당 시장만 돌아갑니다.
- 신경망은 TensorFlow/Keras를 사용합니다. 설치가 부담되면 코드의 `sklearn MLPRegressor` 대체 옵션을 쓸 수 있습니다.
